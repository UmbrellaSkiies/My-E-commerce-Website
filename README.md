## Installation

To install **My-E-commerce-Website**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/UmbrellaSkiies/My-E-commerce-Website.git
```

Windows:

```bash
git clone https://github.com/UmbrellaSkiies/My-E-commerce-Website.git
```
